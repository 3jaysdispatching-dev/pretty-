# Three System

A structured design system workspace for a brand, UI, motion, and marketing ecosystem.

## Sections

- `01_Design_System`: Design principles, components, motion specs, icons, and code libraries.
- `02_Brand_System`: Logo, color palette, typography, voice tone, and photography guidelines.
- `03_UI_Kits`: Ready-made UI kits for mobile, web, and dashboard experiences.
- `04_Motion_Identity`: Motion language, brand animation guidance, and logo reveals.
- `05_Marketing`: Marketing templates, ad creative frameworks, and brand storytelling.

## How to use

Start by reviewing the handbook in `01_Design_System/Handbook` to understand the system foundation. Use the nested folders to document design and developer artifacts, and keep this workspace as the source of truth for your product identity.
