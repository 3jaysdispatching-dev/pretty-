# THREE Brand Identity

**Name**: THREE  
**Full**: Total Human Resource & Route Execution Engine  
**Tagline**: Intelligent Fleet Dispatch Management  

## Logo

### Primary Logo
- Hexagon shape with 3 connection lines
- Gradient: Purple (#9D4EDD) → Pink (#FF006E)
- Cyan accent core (#00D9FF)
- Represents: intelligence, connectivity, autonomy

### Usage
- Minimum size: 32px (digital), 10mm (print)
- Clear space: 8px on all sides
- Never rotate, stretch, or modify
- Always use full color (no modifications)

### Variants
- **Primary**: Full color (preferred)
- **Monochrome**: White/black (minimal)
- **Icon**: Hexagon symbol only (compact)

## Brand Attributes

| Attribute | Specification |
|-----------|---------------|
| Voice | Professional, intelligent, human-first |

| Mission | Simplify fleet management |

## Color Meanings

| Color | Meaning | Use |
|-------|---------|-----|
| Purple #9D4EDD | Intelligence, innovation | Primary CTA, highlights |
| Pink #FF006E | Energy, urgency | Alerts, secondary CTA |
| Cyan #00D9FF | Trust, data | Info, real-time updates |
| Green #06FFA5 | Success, go | Confirmations, online status |
| Orange #FFB703 | Caution, warning | Warnings, attention needed |

## Photography Style

- Modern, tech-focused
- Real fleet operations
- Diverse drivers and backgrounds
- Authentic, not stock-photo staged
- Color-graded with THREE palette

## Typography

**Primary**: Inter (clean, modern)  
**Secondary**: Fira Code (technical data)

Never shorten or modify.
