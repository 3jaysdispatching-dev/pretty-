# Color Usage

Brand color usage defines primary, secondary, neutral, and accent palettes.

## Primary

- Use the primary palette for brand expression, buttons, links, and key UI elements.
- Reserve the strongest primary tone for calls to action.

## Secondary and accent

- Use secondary colors for supporting actions, badges, and charts.
- Use accent colors sparingly to draw attention without overwhelming the interface.

## Neutrals

- Use neutral tones for backgrounds, surfaces, borders, and text.
- Maintain clear distinction between surface layers and input backgrounds.

## Accessibility

- Validate color contrast in each UI state.
- Use dark text on light surfaces and light text on dark surfaces.
