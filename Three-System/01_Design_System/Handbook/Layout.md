# Layout

Strong layout systems create rhythm, structure, and responsive clarity.

## Grid and responsive structure

- Use a flexible column system for content alignment.
- Define breakpoints for mobile, tablet, and desktop.
- Use consistent gutters and margins to keep spacing harmonious.

## Spacing

- Establish a baseline spacing scale: `4px`, `8px`, `16px`, `24px`, `32px`, `40px`.
- Prefer composable spacing utilities instead of hard-coded values.

## Alignment

- Align elements to grid lines for cleaner visual flow.
- Center cards and dialogs within their available layout bounds.

## Layout patterns

Document common page structures such as:

- dashboard panels
- form layouts
- card grids
- split-screen content
