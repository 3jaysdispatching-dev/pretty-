# Icon List

The icon set is organized by function and usage.

## Common categories

- `navigation`: back, close, menu, chevron
- `actions`: add, edit, delete, share, search
- `status`: success, warning, error, info
- `media`: play, pause, camera, download

## Usage notes

- Use icons only when they improve clarity.
- Pair icons with text labels for critical actions.
- Keep icon sizes consistent across related interface elements.
