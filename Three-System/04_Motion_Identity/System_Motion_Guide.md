# System Motion Guide

Motion should support meaning, reinforce hierarchy, and feel intuitive.

## Motion principles

- **Purposeful**: Motion should clarify transitions, focus, and alerts.
- **Subtle**: Keep animations smooth and unobtrusive.
- **Consistent**: Use a small set of timing curves and durations.

## Timing

- Short: `100ms` for micro transitions and feedback.
- Medium: `200ms` for component entrance and hover effects.
- Long: `300ms` for page transitions and larger layout changes.

## Interaction patterns

- Use fade and scale for overlays and dialogs.
- Use slide and opacity for navigation transitions.
- Use motion to guide attention without creating cognitive load.
